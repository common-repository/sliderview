== Description ==
This plugin allows you to create media galleries / slideshows containing videos and photos.

== Installation ==
1. SliderView can be installed directly through your WordPress
Plugins dashboard.
2. Click "Add New" and Search for "SliderView"
3. Install and Activate
Alternatively you can download the plugin using the download button on this page and then upload the sliderview folder to the /wp-content/plugins/ directory then activate throught the Plugins dashboard in WordPress

== Used ==
Copy shortcode to view your slider: [sliderview id="YOUR_ID"]
